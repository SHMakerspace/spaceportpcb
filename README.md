Spaceport PCBs
============
This repos is the 2 PCBs which make up the client portion of [Spaceport](https://github.com/SHMakerspace/spaceport). This consists of the **Spaceport Controller** and the **Spaceport reader**.

Design has been created in EagleCAD.
